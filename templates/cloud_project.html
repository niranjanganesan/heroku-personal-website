COVID-19 Web App that displays the daily stats of Global affected cases

The Global pandemic CoronaVirus (COVID-19) also called as SARS-CoV-2 has already affected millions of people all over the world. The COVID-19 Web App is a prototype of a Cloud Application developed in Python and Flask that gives the global statistics of COVID-19 affected cases. This App makes use of a REST API service to get the daily statistics of COVID-19 and the application is deployed in the AWS EC2 instance.

The applicaion provides the daily updated stats of COVID19 cases on a global scale using the data from External API.

Application features and Infrastructure

⚫ Dynamically generated REST API that has set of services for the selected application domain. The REST API responses conforms to REST standards (response codes).

⚫ The application makes use of an external REST service to complement its functionality: REST API for COVID 19 called COVID19API has been used.

⚫ The application uses Apache Cassandra Cloud database for accessing persistent information.

⚫ Kubernetes Load Balancing service for application deployment.

⚫ Support for Cloud Scalability, deploying in a container environment.

⚫ Cloud Security measures by serving the application over HTTPS using self-signed certificate.

⚫ Request followup orchestration using HATEOAS.

Interacting with Web Application

Interaction with External API

⚫ Route to interact with COVID19API to get the summary of all countries and global stats.

⚫ Route to get the list of all countries along with slug and country code details.

⚫ Route to load the databse with COVID19 country stats using the data from COVID19API


COVID 19 Web App Home Page

The hompage contains links to interact with External API, Load the Cassandra database and view the country wise stats.

REST-based Service Interface

The flask app has been programmed to use the GET, POST, PUT and delete methods of REST API interface to access the information from this Web App.

Some of the REST functions are listed below

⚫ GET @app.route('/')

The Home page displays the Global statistics of COVID 19 along with links to view the Country wise stats and interaction with external REST API.

⚫ GET @app.route('/summary/country')

Get the stats of all countries from the database

⚫GET @app.route('/summary/global')

Get the global stats from the database.

⚫GET @app.route('/summary/country/')

Get the country specific stats from the database.

⚫POST @app.route('/summary/country')

Add a new country to the database. The user must provided the following:
•Country
•NewConfirmed
•TotalConfirmed
•NewDeaths
•NewRecovered
•TotalDeaths
•TotalRecovered

⚫PUT @app.route('/summary/country')

Updates an existing country to the database. The user must provided the following:
•Country
•NewConfirmed
•TotalConfirmed
•NewDeaths
•NewRecovered
•TotalDeaths
•TotalRecovered

⚫DELETE @app.route('/summary/country')

Deletes a country record from the database.

⚫Similar REST functions are implemented to access the global stats from the cassandra table.

App Security, Database and Load Balancing service

App Security

A self signed certificate using TLS encryption is manually configured to run the Flask app over HTTPS

Database Used

Cassandra NoSQL Database is used to store the data from External API

Loadbalancing Service

A Kubernetes Loadbalancing service is configured with 3 replicas to run the application in the cloud service.
